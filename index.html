RECENT WORK

Please feel free to explore the live demos and GitHub repos.

Crypto Connect

Node.js, MongoDB, Express.js, Coinlib API, PassportJS, Tailwind CSS

Crypto Connect is a full stack web application designed to allow users to log in via Coinbase credentials, view live cryptocurrency prices, charts, and detailed coin data, and engage with friends by upvoting or downvoting other crypto trades.

City Spaces NY

Node.js, MongoDB, Express.js, NYC Open Data API, OpenWeather API, PassportJS, Tailwind CSS

City Spaces NY is a full stack web application that is committed to a creating a thriving society, allowing users to search through over 600 volunteer opportunities around NYC, displays the current weather conditions for outdoor service events, add the organizations they wish to volunteer with to a saved list, and facilitate group discussions at the completion of service.

Goddess Styles

CSS, HTML, JavaScript, SCSS

Goddess Styles is a modern, fully responsive, and semantically structured web application for Goddess Styles. This application features a beautiful, custom built user interface that describes the various services provided along with pricing, and allows prospective customers to submit inquiries directly to the stylist.
