NYS Prosecutorial Discovery Checklist

Node.js, MongoDB, Express.js, Tailwind CSS

Discovery Checklist is a full stack web application designed to help legal teams gather discovery, see the types of discovery requests necessary for a case, and advance legal expertise with legal resources. Employees may cretae and log into their account, input the cases they have been assigned to, and keep track of which discovery items are required for trial.

Crypto Connect

Node.js, MongoDB, Express.js, Tailwind CSS

Crypto Connect is a full stack web application designed to allow users to log in via Coinbase credentials, view live cryptocurrency prices, charts, and detailed coin data, and add coins to your portfolio.

City Spaces NY

Node.js, MongoDB, Express.js, NYC Open Data API, Passport, Tailwind CSS

City Spaces is a full stack web application which allows users to search through over 600 volunteer opportunities around NYC, add the organizations they wish to volunteer with to a saved list, and facilitate group discussions at the completion of service.

Goddess Styles

CSS, HTML, JavaScript, SCSS

A modern, fully responsive, and semantically structured web application for Goddess Styles. This application allows prospective customers to submit inquiries directly to the stylist.
